CRITICAL - Valid Mako Syntax ONLY:
- Variables: ${variable_name}
- Conditionals: % if condition:  ... % endif
- Loops: % for item in items: ... % endfor
- Comments: ## This is a comment
- Multi-line Python: <% code here %>
- Define functions: <%def name="func_name(args)"> ... </%def>
- Call functions: ${func_name(args)}

INVALID SYNTAX (DO NOT USE):
- % define / % enddefine (WRONG - does not exist)
- % def / % enddef (WRONG - use <%def> tags instead)
- {% ... %} (WRONG - that's Jinja2, not Mako)

Best Practices:
1. Keep templates simple - prefer direct variable substitution over functions
2. Use ${variable_name} for inserting values
3. Handle missing data with .get(): ${data.get('key', 'default')}
4. Use % if / % endif for conditionals on their own lines
5. Output JSON by writing it directly, not wrapping in functions
